## ImACCESS Project
TAU | Tampere University | Finland

To replicate the results from HistoryCLIP paper: 
1) Download the History-X4 dataset from [here](https://github.com/mrgransky/ImACCESS/tree/main/datasets).
2) For training and evaluation, please refer to the [HistoryCLIP repository](https://github.com/mrgransky/historyCLIP).
